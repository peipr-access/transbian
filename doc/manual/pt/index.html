Resumo

Este documento contém as instruções de instalação para o sistema Debian GNU/Linux 11, (nome de código “bullseye”), para a arquitetura 64-bit PC (“amd64”). Também contém indicações para mais informações e formas de tirar o máximo do seu novo sistema Debian.

Se desejar melhorar, manter, contribuir ou fazer comentários acerca da tradução deste manual entre em contacto com a equipa de tradução em DebianPT.org.
