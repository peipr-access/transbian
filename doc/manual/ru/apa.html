Данный документ описывает установку Debian GNU/Linux bullseye на 64-bit PC («amd64») с помощью нового debian-installer. Это краткое описание процесса установки, в нём содержится вся информация, необходимая для большинства установок. В тех местах, где полезно прочитать более обширное описание, будет приведена ссылка на соответствующую главу этого документа.
