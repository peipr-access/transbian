A.4. Отправьте нам отчёт об установке

Если вы успешно справились с установкой с помощью debian-installer, пожалуйста, найдите время и отправьте нам отчёт. Самый простой путь отправки отчёта — установить пакет reportbug (apt install reportbug), настроить reportbug по описанию из Раздел 8.4.2, «Отправка почты из системы» и запустить команду reportbug installation-reports.

Если завершить установку не получилось, то вероятно вы обнаружили ошибку в debian-installer. Для улучшения программы установки необходимо, чтобы мы узнали об этом, поэтому, пожалуйста, сообщите об ошибке. Вы можете использовать отчёт об установке, чтобы сообщить о проблеме; если установить вообще ничего не удалось, смотрите Раздел 5.4.6, «Сообщение о проблемах при установке».
