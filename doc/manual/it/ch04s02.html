4.2. Recupero dei file dai mirror Debian

Per scoprire qual è il mirror più vicino (e quindi probabilmente il più veloce) si consulti l'elenco dei mirror Debian.

4.2.1. Dove trovare i file per l'installazione

È possibile trovare i vari file per l'installazione su ogni mirror Debian nella directory debian/dists/bullseye/main/installer-amd64/current/images/; in MANIFEST sono elencate tutte le immagini e il loro scopo.
