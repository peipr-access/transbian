Este documento describe cómo instalar Debian GNU/Linux bullseye para 64-bit PC (“amd64”) con el nuevo debian-installer. Es un paseo rápido por el proceso de instalación que contiene toda la información necesaria para la mayoría de las instalaciones. Enlazaremos con explicaciones más detalladas disponibles en otras partes del documento cuando puedan ser útiles para obtener más información.
