4.2. Télécharger les fichiers depuis les miroirs Debian

Pour trouver le miroir le plus proche (et sans doute le plus rapide), voyez la liste des miroirs Debian.

4.2.1. Où trouver les fichiers d'installation ?

Sur tous les miroirs Debian, les fichiers pour l'installation sont situés dans le répertoire debian/dists/bullseye/main/installer-amd64/current/images/ — le fichier MANIFEST liste toutes les images.
