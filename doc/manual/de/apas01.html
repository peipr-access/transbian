A.1. Einleitung

Falls Sie während der Installation auf Fehler stoßen, erhalten Sie im Abschnitt 5.4.7, „Installationsberichte einschicken“ Instruktionen, wie Sie diese an uns melden. Falls Sie Fragen haben, die in diesem Dokument nicht beantwortet werden, richten Sie sie bitte an die Debian-Boot-Mailingliste (debian-boot@lists.debian.org) oder fragen Sie im IRC (#debian-boot im OFTC-Netzwerk) nach.
