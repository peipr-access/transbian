8.5. Einen neuen Kernel kompilieren

Warum sollte jemand einen neuen Kernel kompilieren wollen? Dies ist höchstwahrscheinlich nicht mehr nötig, da der Standard-Kernel in Debian nahezu alle Konfigurationen unterstützt.

Falls Sie jedoch trotzdem einen eigenen Kernel erzeugen möchten: dies ist natürlich problemlos möglich. Wir empfehlen dazu die Verwendung des „make deb-pkg“-Targets. Nähere Informationen hierzu finden Sie im Debian Linux Kernel Handbook.
